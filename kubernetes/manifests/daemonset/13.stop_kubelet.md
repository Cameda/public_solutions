# Отключить kubelet на части нод
```
cat <<EOF | kubectl apply -f -
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: kubelet-stop
  namespace: default
  labels:
    k8s-app: kubelet-stop
  annotations:
    author: cameda
spec:
  selector:
    matchLabels:
      app: kubelet-stop
  template:
    metadata:
      labels:
        app: kubelet-stop
    spec:
      containers:
      - name: kubelet-stop
        image: ubuntu:20.04
        imagePullPolicy: IfNotPresent
        resources:
          requests:
            cpu: "200m"
            memory: "200Mi"
          limits:
            memory: "300Mi"
        command:
          - /bin/sh
          - -c
          - |
            chroot /mnt/cameda /bin/sh -c "sudo systemctl stop kubelet && sleep infinity"
        securityContext:
          privileged: true
        volumeMounts:
        - name: kubelet-stop
          mountPath: /mnt/cameda
      hostNetwork: true
      hostPID: true
      hostname: kubelet-stop
      restartPolicy: Always
      nodeSelector:
        kubernetes.io/os: linux
        test: test
      tolerations:
      - effect: NoSchedule
        operator: Exists
      volumes:
      - name: kubelet-stop
        hostPath:
          path: /
          type: Directory
EOF
```

## Включаем kubelet.
1. Удялем ds kubelet-stop
2. Снимаем с нод uncordon.
