
# Polaris

## Описание.

Polaris позволяет находить дыры в безопасности у установленных в кластере ресурсов.
Все ресурсы группируются по неймспейсам.

### Ресурсы.

* ClusterRole
* ClusterRoleBinding
* Role
* RoleBinding
* ServiceAccount
* Deployment
* StatefulSet
* DaemonSet
* Pod
* CronJob
* Job
* Secret
* ConfigMap
* PDB
* Service
* Ingress
